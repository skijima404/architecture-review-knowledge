## Description
ビジネス部門とアーキテクチャ設計者の間でビジョンの認識が食い違い、実装が期待された価値に結びつかない。

## Context
多くの場合、アーキテクト側がビジネスビジョンを正しく理解していない、または設計に反映していない。  
また、ビジョン自体が曖昧なまま設計が進められるケースもある。

## Impact
実装されたシステムがビジネスの期待に沿わず、再設計や修正のコストが発生し、信頼性や効率性が損なわれる。
